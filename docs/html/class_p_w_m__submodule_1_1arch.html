◆ PWM_signal()

PWM_signal (   PWM_clockscaler ( 2 )  
)
Process

Process der generer PWM signalet. PWM signalet bliver generet ved at cnt signalet tæller op med clocken. Når cnt er lavere end duty_cycle er output benet høj. Når cnt er højere end duty_cycle er output benet lavt. På den måde kan man ved at ændre duty_cycle bestemme hvor meget af tiden PWM signalet er højt og lavt.
